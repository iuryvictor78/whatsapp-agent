## 📋 COMO USAR

### **PASSO 1: Fazer Deploy no Render**

1. Vá para: **https://render.com**
2. Crie uma conta (use GitHub é mais fácil)
3. Vá para: **https://dashboard.render.com**
4. Clique em **"New"** → **"Web Service"**
5. Clique em **"Connect a repository"**
6. Procure por: **"whatsapp-agent"** (seu repositório)
7. Clique em **"Connect"**
8. Preencha:
   - **Name:** `whatsapp-agent`
   - **Environment:** `Node`
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
   - **Plan:** `Free`
9. Clique em **"Create Web Service"**
10. **Aguarde 5-10 minutos** até terminar

Quando terminar, você terá uma URL como: `https://whatsapp-agent-xxxxx.onrender.com`

---

### **PASSO 2: Conectar seu WhatsApp**

1. Abra o **WhatsApp** no seu telefone
2. Vá para **Configurações** → **Dispositivos Conectados** → **Conectar um Dispositivo**
3. Escaneie o **QR Code** que aparecerá no console do Render
4. Aguarde a conexão

**Pronto! Seu WhatsApp está conectado ao agente!** ✅

---

### **PASSO 3: Compartilhar com os Pais**

Agora você pode compartilhar seu número WhatsApp com os dois pais!

**Envie esta mensagem para eles:**

```
Olá! Criei um agente de WhatsApp para facilitar nossa comunicação sobre [NOME DA CRIANÇA].

📱 Número: [SEU NÚMERO WHATSAPP]

Como funciona:
1. Mande uma mensagem para este número
2. O agente pedirá o número do outro responsável
3. Você envia a mensagem
4. O agente analisa se está respeitosa
5. Se ok, encaminha para o outro responsável

Exemplos:

✅ MENSAGENS OK:
"João precisa de novo uniforme"
"Maria tem dentista segunda às 14h"

🔄 MENSAGENS AJUSTADAS:
"Você sempre foi negligente! João precisa de uniforme!"
→ Agente sugere: "João precisa de novo uniforme"

❌ MENSAGENS BLOQUEADAS:
"Você é incompetente!"
→ Agente bloqueia (muito agressiva)

Vamos tentar?
```
